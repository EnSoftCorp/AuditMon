AuditMon { Installation Guide }

Getting started with AuditMon is easy.

Step 1

First make sure you have Atlas Standard or Pro version 2.0.2 or higher installed. For details visit http://www.ensoftcorp.com/atlas/download/

Step 2

1. Start Eclipse, then select Help > Install New Software.

2. Click Add, in the top-right corner.

3. In the Add Repository dialog that appears, enter "AuditMon" for the Name and http://auditmon.com/updates/ for the Location.

4. Click OK.

5. In the Available Software dialog, select the checkbox next to "Code Audit Analysis" and click Next.

6. In the next window, you'll see a list of the tools to be downloaded. Click Next.

7. Read and accept the license agreements, then click Finish. If you get a security warning saying that the authenticity or validity of the software can't be established, click OK.

8. When the installation completes, restart Eclipse.
